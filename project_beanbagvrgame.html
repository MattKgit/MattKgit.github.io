Bean Bag VR Game
XR Terra AR & VR Developer Program

As part of XR Terra AR & VR Developer Certificate Program, we were given the assignment to create a VR Bean Bag Toss game, replete with functionality that kept score for two players.

In this VR experience, created with Unity for the Oculus Quest, a user is situated beside a table and able to move around a small yard with minimal vegetation. The user is able to distance grab bean bags then toss/drop them onto a cornhole board. The score, Red versus Blue, is maintained as the user tosses the bags on the ground, board, or through the hole. After each round, the bean bags are reset upon a table for the user to grab and toss. When a certain score is met, the winner (Red/Blue) is displayed. This can be a two-player game with a shared headset.

Responsibilities

  • Created a two-player VR game using a single Oculus Quest headset
  • Implemented distance grabbing to pick up a bean bag and toss it/drop it
  • Implemented a scoring system, rounds marked when all bags had been tossed, a score condition, and the ability to play again
  • Created a "debug" window to help with debugging issues in VR
  • Played a soothing background music loop, and incorporated sound effects when the bean bags hit the board, the ground, or went through the hole in the corn hole board
